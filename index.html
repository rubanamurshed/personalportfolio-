Projects

Cerebral Blood Flow in Schizophrenia using ASL Perfusion MRI

Investigating cerebral blood flow patterns in schizophrenia using arterial spin labeling (ASL) perfusion MRI. Conducted imaging analyses and processed preliminary results with ASLPrep to better understand neural mechanisms underlying cognitive symptoms.

Tech / Methods: ASL perfusion MRI, ASLPrep, neuroimaging analysis, Python

Predictive Processing in Psychosis (7T fMRI)

Supporting research on predictive processing in psychosis by testing and optimizing a cognitive task performed during 7T fMRI scans. This work aims to understand symptoms such as hallucinations and delusions through neuroimaging and cognitive neuroscience.

Tech / Methods: 7T fMRI, cognitive task design, PsychoPy, neuroimaging

Dual-Valence Pavlovian Conditioning in Mice

Final year undergraduate thesis at the Bagot Lab investigating appetitive and aversive learning in mice using a novel dual-valence Pavlovian conditioning paradigm. Applied machine learning (DeepLabCut) for behavioral analysis to understand risk factors for stress-related disorders.

Tech / Methods: Python, DeepLabCut, Pavlovian conditioning, behavioral neuroscience, chronic social defeat stress

Musician Advantages in Pitch Processing: Brain Connectivity

ARIA-funded research project analyzing how frequency-following responses (FFRs) and functional connectivity differentiate pitch processing in musicians vs. non-musicians. Used MATLAB and Brainstorm for fMRI analysis and functional connectivity analysis.

Tech / Methods: MATLAB, Brainstorm, fMRI, functional connectivity analysis (FCA), FFR

Hippocampal vs. Non-Hippocampal Dependent Memory

Research project at the Coffey Lab exploring differences between hippocampal and non-hippocampal memory systems and the role of sleep events in memory consolidation.

Tech / Methods: MATLAB, behavioral analysis, sleep research

Touchscreen Cognitive Testing in Mice

Research technician work at the CoBrA Lab conducting touchscreen-based cognitive testing in mice, including training, task execution, behavioral monitoring, and data processing using Python and lab-specific analysis pipelines.

Tech / Methods: Python, behavioral neuroscience, animal handling (CMARC-certified), cognitive testing
